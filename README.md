# 🏦 Financial Planner Pro

A sophisticated, modern, and persistent personal finance companion designed for precise budgeting and wealth tracking. Built with React and Vite, this tool follows industry standards for performance, aesthetics, and user experience.

![Preview Image Placeholder](https://via.placeholder.com/1200x630.png?text=Financial+Planner+Pro+Interface)

## ✨ Key Features

- **🏠 Comprehensive Dashboard**: An all-in-one view of your monthly take-home salary, spending, savings, and investment allocations.
- **📈 Dynamic Phase Planning**: Automatically calculates your transition from "Building Emergency Fund" (Phase 1) to "Full Investment" (Phase 2).
- **💾 Local Persistence**: All data is securely saved in your browser's local storage—no account required, and your data never leaves your device.
- **📊 Interactive Visualizations**: SVG-based pie charts and progress bars provide immediate visual feedback on your financial health.
- **🚀 Long-term Projections**: Estimate your wealth over 1-25 years with adjustable CAGR and annual step-up percentages.
- **📅 Monthly Overrides**: Plan for specific months (e.g., insurance premiums or holiday spends) without affecting your default monthly budget.

## 🛠 Tech Stack

- **React**: Modern state management and component architecture.
- **Vite**: Ultra-fast build tool for modern web development.
- **Vanilla CSS**: Premium dark-mode UI with glassmorphism effects and responsive layouts.
- **Lucide Icons**: Crisp, professional iconography.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (v9 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/PK284/financial-planner.git
   cd financial-planner
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

## 🏗 Build for Production

To generate a production-ready bundle:
```bash
npm run build
```
The output will be in the `dist/` directory, ready to be hosted on Vercel, Netlify, or any static hosting service.
